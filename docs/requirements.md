# Requirements

## Functional Requirements

### FR-1 导入文件夹
系统必须支持通过原生文件夹选择器导入目录。

### FR-2 拖拽导入
系统必须支持拖拽单个 MP3 文件、多个 MP3 文件或文件夹到窗口进行导入。

### FR-3 递归扫描
系统必须递归扫描文件夹中的 `.mp3` 文件。

### FR-4 列表展示
系统必须展示导入得到的曲目列表，并支持选中曲目。

### FR-5 播放控制
系统必须支持播放、暂停、上一首、下一首。

### FR-6 进度显示
系统必须显示当前曲目的已播放时间，并在可用时显示总时长。

### FR-7 状态反馈
系统必须提供扫描中、空列表、错误提示等反馈。

## Non-Functional Requirements

### NFR-1 轻量
首版不引入数据库和复杂插件系统。

### NFR-2 响应性
扫描目录时 UI 不能明显卡死。

### NFR-3 稳定性
播放错误和扫描错误必须被捕获并转化为可见状态。

### NFR-4 可维护性
扫描、播放、UI 模块必须可独立理解和修改。
