# PRD — Tips Minimal Local MP3 Player

## 1. 产品定位

Tips 是一个面向本地音频文件的桌面 MP3 播放器。

目标用户不是重度媒体库管理者，而是希望：

- 快速导入本地音乐
- 简单浏览播放
- 用极少的界面完成核心播放操作

## 2. 产品目标

- 在本地桌面上提供一个轻量、稳定、低认知负担的 MP3 播放体验
- 支持拖拽与文件夹扫描
- 在 MVP 阶段只聚焦单窗口使用场景

## 3. 成功标准

- 用户可以在 30 秒内完成导入并开始播放
- 扫描目录不会阻塞界面
- 无需额外配置即可播放本地 MP3
- 视觉风格简洁、克制、接近 mac 工具型应用的体验

## 4. 核心功能

### 4.1 导入

- 通过文件夹选择器导入目录
- 通过拖拽文件 / 文件夹导入
- 自动递归扫描目录中的 `.mp3`

### 4.2 列表

- 展示本次导入得到的本地曲目列表
- 支持单击即播与当前播放项高亮
- 标记当前播放项
- 支持播放列表管理：播放选中项、删除选中项、清空列表
- 删除或清空后，选中态与当前播放态保持一致

### 4.3 播放

- 播放 / 暂停
- 上一首 / 下一首
- 显示当前播放进度
- 显示当前播放曲目标题

### 4.4 状态反馈

- 扫描中状态
- 空状态
- 错误提示

## 5. 非目标

- 在线音乐
- 多格式复杂解码生态管理
- 歌词与封面增强
- 播放列表持久化数据库
- 社交分享
- 高级音频效果

## 6. 用户主路径

1. 打开应用
2. 点击 `Import Folder` 或直接拖拽文件夹
3. 应用后台扫描 MP3
4. 中央列表出现曲目
5. 单击曲目开始播放
6. 顶部可对当前列表执行播放选中项、删除选中项、清空列表
7. 底部栏控制暂停、切歌、查看进度
