④ 交互规则 & 组件说明
A
侧边栏宽度固定 200px macOS traffic lights 区域高度 38px，避让窗口控件
B
list-header sticky + blur position:sticky top:0，backdrop-filter:blur(8px)，z-index:10
C
track-row hover 三态 默认→序号，hover→▶按钮，播放中→eq-bars动画，暂停→⏸
D
键盘快捷键 Space=暂停/播放 · Enter=播放选中 · Del=删除 · ⌘A=全选 · ↑↓=导航
E
playback-bar 高度 80px main-content 末尾 flex 子元素，不是 app-container 子元素
F
云端 CORS 策略 PUT/DELETE 从 Rust 端执行（reqwest）绕过 WebView CORS 限制
G
在线音乐搜索防抖 220ms 搜索框 debounce，平台切换立即清空结果
H
错误/信息消息 info-bar 4s 自动消失，error-bar 6s 自动消失，可手动关闭
I
全屏播放器 z-index:20 overlay，黑胶旋转 CSS animation，点击歌词区域跳转
